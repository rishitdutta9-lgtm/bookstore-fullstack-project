# 📚 Bookstore Fullstack Project

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## Features
- Add Book
- View Books
- Update Book
- Delete Book

## Project Structure
- backend/ → Spring Boot application
- frontend/ → React application

## How to Run

### Backend
Run BookstoreApplication.java

### Frontend
cd frontend
npm install
npm start
